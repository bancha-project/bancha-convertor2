## pageの追加
1. pagesディレクトリ以下に.vueファイルを作成する
1. components.navにアイテムを追加する

## build
ローカルサーバー起動
```
yarn dev
```

ビルド
```
yarn build
```

ビルドしたやつをローカルサーバーで起動
```
yarn start
```

静的サイトを作る（使わない？）
```
yarn generate
```
